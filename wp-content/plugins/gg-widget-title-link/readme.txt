Link the title on any widget.

== Description ==
Lets you link the title of any widget to a URL of your choosing.

== Installation ==
1. Upload `gg-widget-title-link.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. The link field is in the usual widget admin interface on each individual widget.
